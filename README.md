## Overview

SmartTasker is a modern, AI-powered task management application designed to help you organize, prioritize, and complete your daily tasks more efficiently. Leveraging machine learning algorithms, SmartTasker suggests the best times and priorities for your tasks, helping you optimize your workflow effortlessly.


## Features

- AI-driven task prioritization and scheduling
- Intuitive drag-and-drop interface
- Smart reminders and due date predictions
- Task categorization with customizable tags
- Analytics dashboard for tracking productivity trends
- Sync across devices (coming soon)


## Tech Stack

- Frontend: React.js with TypeScript
- Backend: Node.js, Express
- Database: MongoDB
- AI: TensorFlow.js for task prioritization
- Authentication: JWT & OAuth 2.0


## Installation

1. Clone the repository

```bash
 git clone https://github.com/noah58-code/SmartTasker.git
```

2. Navigate to the project directory

```bash
 cd SmartTasker
```

3. Install dependencies for frontend and backend

```bash
 cd client && npm install
 cd ../server && npm install
```

4. Set up environment variables in the `server/.env` file (e.g., database URL, JWT secret)

5. Run the backend server

```bash
 cd server
 npm start
```

6. Run the frontend development server

```bash
 cd ../client
 npm start
```


## Usage

Once running, open your browser and navigate to `http://localhost:3000`. Sign up or log in with your credentials to start creating and managing your tasks. The AI will automatically suggest optimal task priorities and schedules as you add new tasks.
